Blog-Doc Settings

Welcome to the Settings Page of Blog-Doc!
This page will guide you on setting the main configuration to begin using Blog-Doc.

Generate a Data Key
Blog-Doc uses Deta Drive to deliver its content.
The first crucial step is to generate a Data Key to communicate with the Drive of Blog-Doc.
As you can read in the documentation, you can do it from the Builder of Blog-Doc or Blog-Doc's card in Horizon.
Once generated, copy and save it somewhere safe.
Now, from Blog-Doc's Builder, navigate to the Develop tab and the Configuration sub-tab. In the Environment Variables paste the generated Data Key in the field in front of BLOG_DOC_TOKEN and click on Save Changes.

The second step is to fill in the desired and correct values in each field on the form below.
Some settings are enabled by default, some other are automatically added, you'll be able to modify them later on.
Lastly, download the default files from this drive and upload them to the appropriate locations before submitting the form. This will load Blog-Doc with some default content so you can start using it right away. You can later delete these files.
